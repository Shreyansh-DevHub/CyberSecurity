# Network Performance Optimization Lab

## Objective
To enhance network efficiency using QoS, bandwidth management, and load balancing.

## Steps
1. Configured QoS for voice traffic.
2. Applied bandwidth limits on interfaces.
3. Implemented load balancing across gateways.
4. Verified performance improvements.

## Deliverables
- QoS configuration file
- Bandwidth monitoring screenshots
- Load balancing diagram

## Learning Outcomes
- Understand QoS prioritization.
- Learn bandwidth and traffic shaping.
- Practice load balancing for redundancy.
