# Network Security Lab

## Objective
To configure firewall ACLs, establish a VPN tunnel, and document Zero Trust segmentation.

## Steps
1. Created ACLs to allow HTTPS and block other traffic.
2. Configured an IPsec VPN tunnel between two routers.
3. Implemented VLAN-based segmentation for Zero Trust.
4. Verified secure connectivity and blocked unauthorized access.

## Deliverables
- Firewall and VPN configuration files
- Security topology diagram

## Learning Outcomes
- Understand ACLs and VPN encryption.
- Learn Zero Trust segmentation principles.
- Practice verifying secure network connectivity.
