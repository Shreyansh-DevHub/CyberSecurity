# Monitoring & Troubleshooting Lab

## Objective
To perform network diagnostics using ping, traceroute, netstat, and Wireshark.

## Steps
1. Tested connectivity using ping and traceroute.
2. Analyzed routing tables and port activity.
3. Captured packets with Wireshark for inspection.
4. Documented issues and proposed fixes.

## Deliverables
- Command outputs
- Wireshark capture file
- Troubleshooting notes
- Network monitoring diagram

## Learning Outcomes
- Understand network monitoring tools.
- Identify latency and routing issues.
- Practice packet analysis and diagnostics.
