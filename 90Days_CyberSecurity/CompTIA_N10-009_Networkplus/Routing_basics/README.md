# Routing Basics (IPv6)

## Objective
To configure and verify IPv4 and IPv6 routing, NAT, and dual-stack operation.

## Steps
1. Configured IPv4 and IPv6 addresses on router interfaces.
2. Added static routes for both protocols.
3. Enabled NAT for IPv4 traffic.
4. Verified connectivity using ping and route commands.

## Deliverables
- Router configuration files
- Routing table screenshots
- IPv4/IPv6 topology diagram
- Notes on NAT and dual-stack behavior

## Learning Outcomes
- Understand dual-stack routing principles.
- Learn how IPv6 coexists with IPv4.
- Practice NAT configuration and route verification.
