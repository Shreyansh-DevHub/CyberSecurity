# Switching & VLANs Lab

## Objective
To understand VLAN segmentation, trunking, and Spanning Tree Protocol (STP) in a switched network.

## Steps
1. Created VLANs for HR, Finance, and IT.
2. Assigned switch ports to VLANs.
3. Configured trunking between switches.
4. Enabled STP to prevent loops.

## Deliverables
- Switch configuration files
- VLAN topology diagram

## Learning Outcomes
- VLANs isolate broadcast domains.
- Trunking allows multiple VLANs across a single link.
- STP ensures loop-free topology.
