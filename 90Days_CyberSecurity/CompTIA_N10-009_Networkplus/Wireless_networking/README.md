# Wireless Networking Lab

## Objective
To configure and secure a wireless network using WPA2/WPA3 encryption and SSID management.

## Steps
1. Created SSID "NetworkPlusLab" on channel 6.
2. Configured WPA3 security with AES encryption.
3. Enabled DHCP for wireless clients.
4. Verified connectivity and IP assignment.

## Deliverables
- Wireless router configuration file
- Wi-Fi topology diagram

## Learning Outcomes
- Understand wireless security standards (WPA2/WPA3).
- Learn SSID and channel configuration.
- Practice DHCP setup for wireless clients.
