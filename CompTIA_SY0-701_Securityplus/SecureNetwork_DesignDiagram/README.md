# Project 5: Secure Network Design Diagram

## Overview
This project demonstrates a layered security model using a network design diagram.  
The goal is to show how firewalls, DMZs, VPNs, and IDS/IPS systems work together to protect internal resources.

---

## Diagram Components
- **Internet** → Entry point for external traffic.  
- **Firewall** → Filters and blocks unauthorized access.  
- **DMZ** → Hosts public services (web/mail) isolated from LAN.  
- **Internal LAN** → Secure zone for users and internal servers.  
- **VPN Gateway** → Provides secure remote access.  
- **IDS/IPS** → Detects and prevents malicious activity.  

---

## Learning Points
- Layered security reduces risk of compromise.  
- DMZ isolates public services from sensitive internal systems.  
- VPN ensures secure communication for remote users.  
- IDS/IPS adds monitoring and proactive defense.  

---

## Conclusion
This diagram illustrates how multiple security layers work together to protect an organization’s network.  
It highlights the importance of defense‑in‑depth in cybersecurity.
