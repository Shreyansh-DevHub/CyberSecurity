# CompTIA Security+ (SY0‑701) Lab Portfolio

## Overview
This repository documents my hands‑on Security+ labs completed on Kali Linux and Windows.  
Each project demonstrates a key cybersecurity concept — from authentication and encryption to network defense and identity protection.  
Together, they form a complete practical portfolio aligned with the CompTIA Security+ SY0‑701 domains.

---

## Project Index

| Project | Folder | Focus Area |
|----------|---------|------------|
| **1. Password Hashing Demo** | `password_hashing` | Authentication fundamentals — hashing vs plaintext |
| **2. Encryption & Decryption Lab** | `encryption_decryption` | Cryptography — symmetric & asymmetric encryption |
| **3. Network Security Basics** | `Network_security_basics` | Firewall rules, port blocking, and access control |
| **4. Vulnerability Scanning** | `Vulnerability_Scanning` | Threat detection and mitigation using scanning tools |
| **5. Secure Network Design Diagram** | `SecureNetwork_DesignDiagram` | Layered defense architecture (DMZ, LAN, VPN, IDS/IPS) |
| **6. Network Monitoring & Troubleshooting** | `Network_Monitoring_&_Trouble...` | Log analysis and network diagnostics |
| **7. Multi‑Factor Authentication (MFA) Simulation** | `MFA_demo` | Identity & access management — password + OTP |
