# 🔐 Project 2: Encryption & Decryption Lab

## 📌 Overview
This project demonstrates two important cryptography techniques:
- **AES (Fernet)** → Symmetric encryption, where the same key is used to encrypt and decrypt.
- **RSA** → Asymmetric encryption, where a public key encrypts and a private key decrypts.

By running these scripts, you’ll see how plaintext messages are transformed into ciphertext and then successfully decrypted back to their original form.

---

## 🛠️ Setup
1. Make sure Python 3 is installed on Windows.
2. Install the required libraries:
 
   "bash pip install cryptography pycryptodome"
